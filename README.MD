# dectobin - A simple program to convert decimal representations into binary ones (base-2). #

##Installation##
### Simply run the Makefile include with command "make" and the program will compile into dectobin###

##Usage##
###To use, simply write `dectobin <uint 0-255>` putting your unsigned decimal integer (current possible range is 0-255) where `<uint 0-255>` is.###
